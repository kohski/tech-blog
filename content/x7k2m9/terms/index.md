本規約は、kohski（以下「当事業者」）が提供するバックエンドエンジニア向けメンタリングサービス（以下「本サービス」）の利用条件を定めるものです。お申し込みにあたっては、本規約に同意いただいたものとみなします。

## 第1条（サービス内容）

本サービスは、Go / TypeScript / JavaScript / Python、コンテナベース・サーバーレスのアーキテクチャ、および AWS に関するバックエンド開発の**メンタリング**を提供するものです。具体的形式（チャット、ビデオ、レビュー回数の目安等）は、当事業者が別途定める運用方針または個別のすり合わせに従います。

## 第2条（契約の成立）

利用者が当事業者が指定する申込手続きを完了し、**初回の決済が Stripe を通じて正常に完了した時点**で、本サービスに関する利用契約が当事業者と利用者の間に成立します。

## 第3条（利用料金）

- **月額プラン**：税込 **¥300 / 月**（以下「利用料」といいます）。
- 料金は当事業者が本サイト上で表示する内容を優先し、法令に反しない範囲で改定されることがあります。既に成立している契約の利用料については、改定内容を別途通知した場合を除き、原則として改定前の条件が適用されます。

## 第4条（支払方法）

利用料の支払いには、**Stripe によるクレジットカード決済**を利用します。カード情報は Stripe のシステム上で処理され、当事業者がクレジットカード番号そのものを保存することはありません。

## 第5条（解約）

- 利用者は、**いつでも解約の申し出**を行うことができます。解約の手続きは、当事業者が指定する方法（決済のカスタマーポータル、メール等）に従ってください。
- 解約後は**次回以降の請求は発生しません**。
- **日割りによる返金は行いません。** 既に支払いが完了している請求期間の末日までは、本規約に従い本サービスを利用できます。当該期間の満了をもって本サービスの提供は終了します。

## 第6条（禁止事項）

利用者は、以下の行為をしてはなりません。

1. 法令または公序良俗に違反する行為
2. 当事業者、他の利用者または第三者の権利を侵害する行為
3. 虚偽の情報を申込時に提供する行為
4. 本サービスを通じて入手した情報を、当事業者の事前の承諾なく第三者に開示・譲渡する行為
5. 本サービスの運営を妨害する行為、不正アクセス行為、第三者になりすます行為
6. その他、当事業者が不適切と判断する行為

当事業者は、禁止事項に該当する行為があった場合、事前の通知なく利用契約を解除し、または本サービスの提供を停止することができます。

2. 本サービスの利用に起因して利用者に生じた損害について、当事業者に故意または重過失がある場合を除き、当事業者は賠償責任を負いません。なお、当事業者が負う賠償責任の上限は、当該損害が発生した月に利用者が支払った利用料の額を上限とします。
