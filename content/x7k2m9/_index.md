## これはなに

**バックエンドエンジニア向けのメンタリング**です。  
コードレビューや設計の相談、キャリアの話まで、月額で伴走します。β のため、提供形式や回数は変更になることがあります。

## カバーする技術領域

| 区分 | 内容 |
| --- | --- |
| **言語** | Go、TypeScript、JavaScript、Python |
| **アーキテクチャ** | コンテナベース、サーバーレス |
| **クラウド** | AWS |

インフラ寄りの設計、API・バッチ、可観測性、コストやセキュリティの話題など、バックエンド周りを中心に扱います。

## サービスの流れ

1. **入会**: 月額プランに登録する。
2. **すり合わせ**: いまの課題やスタック、相談したい粒度を共有する。
3. **メンタリング**: チャットやビデオなど、運用ルールに沿ってやり取りする。
4. **継続**: 翌月以降も、開発のフェーズに合わせて伴走する。

※ 具体的な形式（同期／非同期、回数の目安など）は β 運用のなかで調整します。

## こんな方におすすめ

- バックエンドを担当しているが、設計や運用の判断に自信が持てない方
- コンテナやサーバーレス、AWS を組み合わせた構成を詰めたい方
- Go / TS（JS）/ Python のいずれかで開発しており、レビューや観点をほしい方
